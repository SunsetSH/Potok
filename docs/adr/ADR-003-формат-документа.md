# ADR-003: Формат rich-document — Quill Delta в версионированном конверте

Статус: принят (2026-07-16)

## Решение
Canonical-формат `Note.document_json`:

```json
{
  "schema": "potok.document",
  "version": 1,
  "format": "quill-delta",
  "delta": { "ops": [ ... ] }
}
```

- Редактор: `flutter_quill`. Delta покрывает paragraph, bold, italic, strike, checklist, link, embeds.
- Изображение — embed `{"image": "asset://<asset_id>"}` + атрибуты alt/display; физически managed file (`MediaAsset`), не base64 (FR-DOC-003).
- Аудио — embed-ссылка `{"audio": "asset://<asset_id>"}`.
- `document_plain_text` — детерминированная проекция Delta для FTS и card preview; пересчитывается при каждом сохранении в той же транзакции.

## Обоснование
Delta — стабильный, широко используемый JSON AST; готовый редактор экономит недели против собственного. Конверт с `version` даёт путь миграции, включая уход с Delta (FR-DOC-001: versioned JSON AST — выполняется).

## Ограничения
- Import/export HTML/Markdown — только через санитайзер (FR-DOC-005): allowlist тегов/схем URL, лимиты size/depth.
- Raw HTML никогда не источник истины.
- Checkbox toggle — обычная правка документа: транзакция, undo, `revision++` (FR-DOC-004).
