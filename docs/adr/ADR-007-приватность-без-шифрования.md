# ADR-007: Приватность — local-only, без шифрования на уровне приложения

Статус: принят (2026-07-16, контракт ТЗ 0.10)

## Решение
- Приложение НЕ шифрует БД, аудио, изображения, модели, backup. Опора: app sandbox + системное шифрование устройства. Без app PIN/biometric (ложное чувство защиты).
- First-run и backup UI явно сообщают: файлы читаемы при доступе к устройству/архиву.
- Нет телеметрии содержимого, нет автоматической отправки крешей с заметками.
- Логи не содержат текст, transcript, media, имена проектов/тегов, пути пользователя, токены.
- Untrusted input (import, backup, document JSON, model pack): лимиты size/depth/path/ratio, параметризованный SQL, allowlist сортировок/колонок.
- Базовая сборка — без сетевых вызовов, кроме загрузчика ASR-моделей: отдельный allowlisted-клиент (`AsrModelManager.downloadAndInstall`), только по явному действию пользователя, только хосты github.com/*.githubusercontent.com. Inference обязан работать в airplane mode — распознавание никогда не обращается к сети.

Шифрование из приложения A (AES-GCM/Argon2id) отклонено редакцией 0.3 ТЗ.
