## Известные обходы/долги
- **Кириллица в пути ломает Dart tooling** («package_config did not contain its own root package»). Митигация применена: junction `h:\.pet\potok` → все flutter/dart-команды через него.
- Срез пишет WAV 16 kHz mono (чтобы ASR работал без декодера); переход на AAC/M4A + decode-шаг — WP-03 (ADR-005).
- В консольном `dart run` System32-овский старый `onnxruntime.dll` перекрывает библиотеку пакета — в spike добавлен preload по абсолютному пути (`SHERPA_DLL_DIR`); в Flutter-приложении проблемы нет (DLL рядом с exe).
- Model manager (паки, hash, активация) — WP-03; сейчас модель берётся из `C:\dev\models\sherpa-onnx-whisper-tiny` (override `POTOK_ASR_MODEL_DIR`).
- Android-сборка не проверена (SDK ставился в фоне) — первый пункт WP-01.
